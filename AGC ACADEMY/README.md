# AGC Academy Website

This project is a single-page, responsive portfolio-style website that showcases **AGC Academy**—a Christian education institution based in Kaptembwo, Nakuru. The experience uses minimalist styling, glassmorphism-inspired cards, gradient accents, and animated elements while remaining lightweight, accessible, and mobile-friendly.

## Highlights
- Full “Home / About / Programs / Contact” layout with sticky navigation, hero statistics, values section, parental testimonials, and CTA-rich footer.
- Programs section covers Daycare Services, ECD, and CBC Primary Education.
- Contact area lists school details, includes social links (GitHub, LinkedIn, Twitter, Facebook community), and keeps the logo visible alongside a location link.
- Dark/light theme toggle plus smooth section reveal, animated stats, and polished button effects.

## Files
- `index.html` – page structure and content.
- `css/style.css` – theming, layout (Flexbox/Grid), gradients, responsive rules, and utility classes.
- `js/script.js` – theme toggle, reveal effects, statistic animations, rotating news, testimonials carousel, sticky nav/back-to-top logic.
- `images/` – contains brand and illustration assets (logo, profile, program placeholders).

## Additional Notes
- The site avoids a full embedded map for reliability but links directly to the Kaptembwo Google Maps pin.
- The footer and hero both reuse the AGC Academy crest, keeping the logo visible without redundant imagery.

Feel free to update the assets or copy to match upcoming events or programs; the layout supports additional sections as needed.
